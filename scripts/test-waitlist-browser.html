Test Waitlist Submission

Open this page at: http://localhost:9009/

Then use the waitlist form on the homepage to test email submission.

Things to verify:

  1. Submit an email address in the waitlist form
  2. Check the browser console for any errors
  3. Check the terminal running `pnpm dev` for email logs
  4. If email sending is disabled, you should see the email content in the console
  5. The email should have "Waitlist Confirmation" as the heading
  6. The verification URL should be http://localhost:9009/api/waitlist/verify-email-token?token=...

If you get "already subscribed" error:

  • Use a different email like: test-${Date.now()}@example.com
  • Or go to: http://localhost:9009/dashboard/admin/waitlist
